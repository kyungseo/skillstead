# Install

## Claude Code

1. Choose your Claude Code skills directory.
2. Copy the skill folder (e.g. `skills/svg-infographic/`) into that directory.
3. If the skill does not appear, restart Claude Code.
4. Invoke it by name, or ask for a matching task.

This doc uses a `<claude-skills-dir>` placeholder instead of a machine-specific absolute path. Do not hard-code a personal home directory in shared docs.
